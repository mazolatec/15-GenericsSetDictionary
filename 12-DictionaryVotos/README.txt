Na contagem de votos de uma eleição,
são gerados vários registros
de votação contendo o nome do candidato e a quantidade de votos(formato .csv) 
que ele obteve em uma urna de votação. Você devefazer um programa para ler os registros de votação a partir de umarquivo, 
e daí gerar um relatório consolidado com os totais de cadacandidato.

Input file example:
Alex Blue,15
Maria Green,22
Bob Brown,21
Alex Blue,30
Bob Brown,15
Maria Green,27
Maria Green,22
Bob Brown,25
Alex Blue,31

Input file example:
Enter file full path: d:\votos.txt
Alex Blue: 76
Maria Green: 71
Bob Brown: 61
